# Plot Image Slots

Place your plot images in:

```text
assets/plots/
```

Use these filenames:

```text
plot-1.png
plot-2.png
plot-3.png
```

The page uses those same three files in two ways:

- faint decorative background cards behind the page
- visible plot cards in the Research section

PNG, JPG, or WebP are fine, but if you use JPG/WebP, update the filenames in `index.html` and `styles.css`.
